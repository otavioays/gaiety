# GAIETY Classic

Landing page editorial para o relógio GAIETY Classic.

## Estrutura visual

A experiência usa a gramática de sites de alta relojoaria: navegação minimalista, imagem em tela cheia, tipografia editorial, seções cinematográficas, cards visuais e uma oferta integrada à narrativa.

A identidade, os textos e os ativos são próprios da GAIETY. Nenhuma marca ou ativo da Rolex está incluído.

## Checkout

No arquivo `script.js`, substitua:

```js
const checkoutUrl="https://seu-checkout.com";
```

pelo endereço real do checkout.

## Publicação

O workflow em `.github/workflows/pages.yml` publica automaticamente o conteúdo da branch `main` no GitHub Pages.
